# Planner Product Refactor Plan

## Objective
Transform the planner into a premium work management module that competes with paid tools.

## Product direction
- modern enterprise header
- KPI/overview strip
- workspace switcher with favorites
- richer kanban cards
- better list view with compact density
- analytics and workload surfaces
- stronger empty/loading states
- faster task actions without opening full dialog for everything

## Refactor phases

### Phase 1 — Shell and navigation
- planner command header
- KPI strip
- view switcher redesign
- plan tabs redesign
- quick actions and import/export grouped

### Phase 2 — Core surfaces
- premium kanban columns
- rich task cards
- better list/table view
- side insights rail

### Phase 3 — Work intelligence
- today / week / overdue smart slices
- assignee workload
- due date risk states
- progress and completion rates

### Phase 4 — Competitive differentiation
- templates
- recurring tasks
- timeline/calendar view
- saved views and favorites

## Design language
- enterprise / modern
- strong hierarchy
- high information density without clutter
- clear state colors
- premium micro-interactions
